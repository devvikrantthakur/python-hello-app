# Python Hello App

A simple Python Flask application deployed on AWS EC2.

## Features

- Built with Python and Flask
- Simple web application
- Deployed on AWS EC2
- Easy to run and test

## Technologies Used

- Python 3
- Flask
- AWS EC2
- Linux

## Project Structure

```
python-hello-app/
├── app.py
├── requirements.txt
└── README.md
```

## Installation

Clone the repository:

```bash
git clone https://github.com/your-username/python-hello-app.git
```

Go to the project directory:

```bash
cd python-hello-app
```

Install the dependencies:

```bash
pip3 install -r requirements.txt
```

Run the application:

```bash
python3 app.py
```

The application will be available at:

```
http://localhost:5000
```

## Application Endpoints

| Endpoint | Description |
|----------|-------------|
| `/` | Displays a success message |
| `/hi` | Displays a simple HTML greeting |
